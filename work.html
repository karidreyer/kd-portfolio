Meet - The Event Finder App

Meet is a serverless, progressive web application (PWA) built with React, utilizing a test-driven development (TDD) technique. The app allows users to explore upcoming events, filtering events by city. It also includes offline support, home screen shortcuts, and information visualizations for a better user experience.

Movie Nest

The Move Nest App is a single-page, responsive app with routing, rich interactions, several interface views, and a polished user experience. The Movie Nest app is designed as a client-side piece to support the server-side Movie Nest API, which was built to facilitate user requests and render the response from the server-side via a number of different interface views.
