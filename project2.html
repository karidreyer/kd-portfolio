Meet - A React PWA

Meet is a serverless, progressive web application (PWA) built with React, built using test-driven development (TDD). The app allows users to view details about upcoming events from the Google Calendar API, search for events by city, adjust the number of events they want to display, and view the types of events in graph visualizations.

Features

  • Filter Events by City.
  • Show/Hide Event Details.
  • Specify Number of Events.
  • Use the App When Offline.
  • Add an App Shortcut to the Home Screen.
  • Display Charts Visualizing Event Details.

Technologies Used

  • React: The app is a React application, enabling a dynamic and component-based architecture.
  • Test-Driven Development (TDD): The app was built using the TDD technique to ensure reliability and maintainability.
  • Google Calendar API: Used to fetch event data, with OAuth2 authentication flow for secure access.
  • AWS Lambda: Serverless functions are utilized as the authorization server, replacing traditional servers.
